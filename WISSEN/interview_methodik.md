## Outside-In Prinzip (konzentrische Kreise)

IMMER von aussen nach innen. Nie den Hauptbeschuldigten zuerst befragen.

```
Phase 1: NEUTRAL — Fakten sammeln
  → IT, Buchhaltung, Prozess-Verantwortliche
  → Ziel: Ablaeufe verstehen, Dokumente sichern

Phase 2: KORROBORATIV — Indizien erhaerten
  → Peripherie-Zeugen, Kollegen
  → Ziel: Alibis pruefen, Netzwerk identifizieren

Phase 3: MITWISSER — Kooperation gewinnen
  → Insider mit Wissen/Bauchschmerzen
  → Ziel: Belastungsmaterial gegen Haupttaeter

Phase 4: ZIELPERSON — Konfrontation
  → Hauptbeschuldigter(e)
  → Ziel: Gestaendnis oder verwertbare Aussage
```

## Fragetechniken

### Kooperative Zeugen (Menkal, Likic, Jascha)
- **Trichter-Technik:** Offene Fragen → W-Fragen → Details
- **Kein "Warum"** (macht defensiv) → stattdessen "Wie kam es dazu?"
- **Echo-Technik:** Letzten Satz wiederholen, animiert zum Weitersprechen

### Blockierende Zeugen (Zippel, Eierdanz)
- **Geschlossene Fragen:** Auf Version festnageln die spaeter widerlegbar ist
- **Korrektur-Technik:** Bewusst falsche Behauptung → Zeuge korrigiert (gibt echte Details preis)
- **Umgekehrte Chronologie:** Erhoehte kognitive Last bei Luegnern

### Insider-Gewinnung (Maage, evtl. Ducree)
1. **Rationalisierung:** "Der Druck war zu gross, oder?"
2. **Minimierung:** Moralische Schuld runter (nicht rechtliche!)
3. **Keil:** "Der Haupttaeter benutzt dich als Suendenbock. Kooperation = Schadensbegrenzung"
4. **Sofort fixieren:** Schriftliche Aussage bevor Meinung sich aendert

### Abschlussinterview — "Admission-Seeking Interview" [E-07] (Bierau, dann Caspari)
1. Direkte Konfrontation ohne Einleitung
2. Einwaende unterbrechen: "Nicht OB — WARUM"
3. Zwei Szenarien: Gier (A) vs. Notfall (B) → meisten waehlen B = Gestaendnis
4. Details sichern: Wie viel? Wo? Wer noch?
